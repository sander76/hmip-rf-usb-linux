# HMIP-RF-USB on Linux

## Driver installation

- Clone this repo
- Go to the `driver` folder.
- enter `. install.sh`

Verify installation has succeeded by checking for a ttyUSB0 device in the `/dev` folder.

## crRFD server

This requires the -not included- server java binary.
